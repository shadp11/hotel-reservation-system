# Hotel Reservation System

The Hotel Reservation System is a Java-based application developed using NetBeans. It manages hotel operations such as user login, guest reservations, room tracking, and housekeeping management using ArrayList data structures. This project demonstrates core Object-Oriented Programming (OOP) concepts and basic data management using a database.

---

## Features

- Multiple user login
- Guest registration and management
- Room booking and availability tracking
- Reservation management
- Payment processing

---

## Tech Stack

- Java (SDK 17)
- Netbeans IDE
- MySQL
- Java Swing

---

## How to Run

1. Clone or download the project
2. Open the project in **NetBeans IDE**
3. Ensure JDK 17 is installed and selected
4. Set the main class:
   ```
   multipleloginarraylist.MultipleLogin
   ```
5. Press **F6** to run the project
